📌 Overview

This repository contains an interactive Power BI dashboard built to analyze Blinkit-related data using clear visualizations, KPIs, and dynamic filters. The dashboard focuses on transforming raw data into meaningful insights that help understand business performance and trends at a glance.

🎯 Project Objective
Analyze key business metrics using Power BI
Present insights through an intuitive and interactive dashboard
Enable data-driven decision-making through visual analytics
Practice real-world dashboard design and storytelling
🛠 Tools & Technologies
Microsoft Power BI Desktop
Data Modeling
DAX (basic measures)
Interactive visuals (cards, bar charts, line charts, slicers)
📊 Key Features
KPI cards highlighting important metrics
Interactive slicers for dynamic analysis
Trend and comparison visualizations
Clean, structured, and user-friendly layout

🚀 How to Use

Clone or download this repository
Open Blinkit_Dashboard.pbix using Microsoft Power BI Desktop
Use slicers and filters to explore insights interactively
⚠️ GitHub does not preview .pbix files. Please download the file to view the dashboard.

💡 Key Insights (Example)

Identification of high-performing and low-performing categories
Understanding trends over time using visual comparisons
Quick evaluation of overall performance through KPIs
